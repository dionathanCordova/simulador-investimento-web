## :checkered_flag: Starting ##

```bash
# Clone this project
$ git clone https://github.com/dionathanCordova/simulador-investimento-web

# Access
$ cd simulador-investimento-web

# Install dependencies
$ yarn

# Run the project
$ yarn start

# The server will initialize in the <http://localhost:3000>
```
